# SiLa-Challenge : Découverte Des Notions Avancées et Bibliothèques.

Ce dépôt contient mes solutions au challenge SiLa.
ceci est un defi de 30 jours me permettant de prendre en main les concepts avancés et de differentes bibliothèques de la programmation en python.

## Objectif
Améliorer mes compétences en programmation en decouvrant plusieurs bibliothèques & des notions avancées en python.

## Organisation
Chaque jour contient :
- Un dossier JourX
- Un fichier README explicatif (Decrit notre solution)
- Le code source correspondant
